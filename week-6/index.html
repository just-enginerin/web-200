    5 Things I've Learned Since the Course Started

  • Something to consider with web content is line length. Depending on the content and audience, do they need to read it quickly? Or would it be better to satisfy the preferences of the user?
  • "Words in All Caps are Difficult to Read" is a common misconception. Using all caps is a great way to capture reader attention, for example, with headlines or alerts.
  • One con of using a user interface framework is that unused code within the framework will likely reside within the project, which adds to loading time.
  • Even extremely vivid and dramatic memories are remembered incorrectly.
  • The structure of an interface can be broken down into a growing number of increasingly smaller pieces, similarly to how all matter in the universe can be broken down into atoms. Atomic Design involves building interfaces so that one small change to one or just a few of the pieces can effectively change the entire interface.
